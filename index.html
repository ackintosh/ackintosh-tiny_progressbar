Ackintosh::TinyProgressbar

My simple gem

Installation

Add this line to your application's Gemfile:

gem 'ackintosh-tiny_progressbar'

And then execute:

$ bundle

Or install it yourself as:

$ gem install ackintosh-tiny_progressbar

Usage

atp = Ackintosh::TinyProgressbar.new(100)
100.times do
  atp.scceed
  sleep 0.1
end
atp.clear
